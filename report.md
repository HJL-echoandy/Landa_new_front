Landa 是一个基于 React Native 的移动端按摩服务预约应用，提供完整的用户注册、服务预约、支付、订单跟踪等功能。

## 技术栈概览

### 核心框架
- **React Native**: 跨平台移动应用开发框架
- **TypeScript**: 类型安全的JavaScript超集
- **Expo**: React Native 开发工具链和平台

### UI框架与组件库
- **Gluestack UI**: 现代化的React Native UI组件库
- **@expo/vector-icons (Ionicons)**: 图标库
- **React Native 内置组件**: SafeAreaView, ScrollView, TouchableOpacity等

### 导航系统
- **@react-navigation/native**: React Native导航核心库
- **@react-navigation/native-stack**: 原生栈导航器
- **@react-navigation/bottom-tabs**: 底部标签导航器

### 状态管理
- **Redux Toolkit**: 现代化的Redux状态管理
- **React Redux**: React与Redux的绑定库
- **Redux Persist**: 状态持久化

### 开发工具
- **开发者菜单增强**: 自定义开发工具，支持快速页面跳转
- **路由发现系统**: 动态路由管理和分类
- **响应式工具**: 自适应屏幕尺寸的工具函数
